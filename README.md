# bigburger

Biiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiiig burger!

![sample](./sample.gif)

## Install

```
$ go get -u github.com/Doarakko/bigburger
```

## Usage

```
Usage of bigburger:
  -C int
    	how many cat
  -R	flip vertical
  -b int
    	how many bun
  -c int
    	how many cheese
  -i float
    	rate of intervals (default 1)
  -l int
    	how many lettuce
  -n int
    	number of big burger (default 1)
  -o string
    	output image file
  -p int
    	how many putty (default 1)
  -r	flip holizontal
  -t int
    	how many tomato
```

## Example

```
$ bigburger -n 3 -p 6 -c 5 -l 3 -t 3
```
